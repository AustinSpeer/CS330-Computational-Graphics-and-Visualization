  This project shows my ability to design and build 3D scenes using OpenGL. Over the time this course took, I moved my focus on only getting objects to render to thinking more about the structure, and long-term maintainability. 

  When I approached the design, I started by breaking the project into manageable pieces. I try to understand what needs to exist before thinking about how it will be implemented. I first decided what objects would make up the environment and how they should be positioned in relation to each other. I had to think about the scale of the objects, camera placement, lighting direction and how the overall layout should feel. Designing 3D space forced me to think differently. I had to visualize how transformations would affect each object and how multiple shapes could combine to form a single cohesive model. 

  This project helped me develop stronger spatial reasoning and a better understanding of how transformations work together, I got more comfortable working with scale, rotation and translation. I gain experience applying textures in a way to not distort geometry. Learning how lighting interacts with material properties strengthened my understanding of how visual realism is created in a rendering pipeline. 

  My design process remained consistent throughout the milestones. I identified one object at a time and broke it down into primitive shapes. I positioned each component applied color to the texture and tested the results. I did not move forward until each part was correct. Working incrementally helps me from having serious issues later. I also made debugging easier because I always know what has changed recently. 

  Computer science helps me reach my goals by training me to think clearly and solve problems step by step. This graphics project forced me to apply math in real code through transformations, lighting, and shaders instead of just learning concepts on paper. It strengthened my foundation for future classes which will help in client server development or mobile architecture. Working with transformations, shaders, and low-level rendering has helped how I think about the structure of the system. Even if I don’t work with graphics professionally this project has helped me handle complex problems step by step.
